# Image Archiver (WIP)
*An image categorisation and archiving desktop application built with PyQt6.*

## Overview
Image Archiver helps you organise, categorise, and archive large sets of images.  
It can automatically tag images using AI object detection (YOLO), streamlining manual sorting.

## Features
- Browse folders of images and view thumbnails
- Automatically assign categories using YOLO
- Move/copy images into structured archive folders based on metadata or rules (WIP)
- Designed for workflows involving large image collections

## Installation & Running
```bash
git clone https://github.com/mkow7045/image-archiver.git
cd image-archiver
pip install PyQt6 ultralytics
python main.py
```
